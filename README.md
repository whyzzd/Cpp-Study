### 基本介绍

这是一个初学者(作者)的仓库，目的是为了整理巩固自己所学基础知识，如果能还能帮到其他人那更好，所有的代码以及注释都是纯手动敲的，所以难免某些地方会有一些错误或者不合理的地方欢迎各位Issue以及Pull request,如果喜欢的话也可以⭐️或者Fork。
